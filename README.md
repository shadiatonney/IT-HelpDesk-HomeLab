# IT Help Desk & Network Engineering Home Lab

Welcome to my **IT Help Desk and Network Engineering Home Lab Portfolio**.

This repository contains hands-on labs demonstrating practical skills in **networking, server infrastructure, virtualization, security, and troubleshooting**.  
The labs simulate **real-world enterprise IT environments** and common **IT Help Desk scenarios**.

---

## Skills Demonstrated

• Network infrastructure design  
• Router configuration and routing protocols  
• Network troubleshooting techniques  
• Server deployment and management  
• Network security planning  
• Virtualization technologies  
• IT support problem solving
